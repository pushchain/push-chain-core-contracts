# Deployment Addresses

This folder contains deployment addresses for each chain.

## File Naming Convention
Files are named by chainId:
- `1.json` - Ethereum Mainnet
- `8453.json` - Base
- `137.json` - Polygon
- `42161.json` - Arbitrum One
- `10.json` - Optimism
- etc.

## File Format
Each JSON file contains:
```json
{
  "chainId": 1,
  "deployer": "0x...",
  "ceaImplementation": "0x...",
  "ceaProxyImplementation": "0x...",
  "ceaFactoryImplementation": "0x...",
  "ceaFactoryProxy": "0x...",
  "owner": "0x...",
  "vault": "0x...",
  "universalGateway": "0x..."
}
```

## Important
- **Always use the Proxy addresses** for contract interactions
- Implementation addresses are for verification and upgrade reference only
- These files are auto-generated by deployment scripts
